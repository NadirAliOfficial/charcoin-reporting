# CharCoin Insights

Analytics, reporting, and global donation tracking dashboard for the CharCoin ecosystem.

## Features
- Token holder analytics
- Donation tracking by region
- Transaction volume charts
- CSV export for reports

## Tech Stack
- JavaScript / Chart.js
- REST API integration
- Bootstrap 5

## Setup
```bash
npm install
npm start
```
